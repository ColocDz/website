# Website for Colocation (for Algerian users)

---
## 1.Problem
There is a need to find trustworthy people to share housing with, while also securing the best prices in the locations where users want to live.

---

## 2.Solution
A website designed for people searching for colocation opportunities. It will offer fast search functionality and secure login using the national ID card.

---

## 3.MVP features
    - Posting new colocation offers with photos of the house, apartment, or individual rooms.
    - Publishing requests to find a roommate and a place to live.
    - Sending private messages (only if the recipient accepts).
    - Deleting or editing any published posts.
    - Authentication using the national ID card, with an age restriction (19+).
    - Advanced search filters: age, gender, price, type of housing, location, wilaya, number of rooms, and number of roommates.
---

## 4.Tech stack
 React + Vite + Router + Context + Node + Express + MongoDB + Mongoose
